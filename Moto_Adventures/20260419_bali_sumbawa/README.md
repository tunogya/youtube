# 🏍️ Bali to Sumbawa Moto Adventure: 项目企划与后勤

本项目记录从巴厘岛出发，穿越龙目岛，最终抵达松巴哇岛（Sumbawa）的史诗级摩托车骑行与摄影探险之旅。本文件主要包含项目的行程路线、后勤物资、预算以及补给策略。内容脚本请参考 `script.md`，拍摄分镜请参考 `shot_list.csv`。

## 🗺️ 1. 地图与路径设计 (Route Design)

**总路线主轴**：Bali (Canggu) -> Padang Bai (轮渡) -> Lombok (Lembar -> Kayangan 轮渡) -> Sumbawa (Poto Tano -> Maluk -> Lakey Peak / Mount Tambora)
**预计时间**：7-10 天

- **阶段一：巴厘岛撤离与跨海** 
  - 路线：Canggu -> Padang Bai 港口
  - 重点：沿东海岸骑行，通过滚装船 (Ro-Ro Ferry) 抵达龙目岛 Lembar 港。
- **阶段二：穿越龙目岛**
  - 路线：Lembar -> Mataram外围核心公路 -> 经由中部高地或东部平原 -> Kayangan 港口
  - 重点：感受龙目岛相对巴厘岛逐渐狂野的景观变化，搭乘第二段轮渡前往松巴哇。
- **阶段三：深入松巴哇（西海岸秘境）**
  - 路线：Poto Tano 港口 -> Kenawa Island (弃车乘小船短暂登岛航拍) -> Maluk / Sekongkang (绝美白沙滩和冲浪圣地)
- **阶段四：狂野东部与坦博拉火山 / Lakey Peak**
  - 路线：Sekongkang -> Sumbawa Besar -> Dompu -> Lakey Peak (Hu'u) 
  - 重点：无尽的沿海公路，荒芜的热带大草原地形。

## 📅 2. 大致行程安排 (Itinerary)

- **Day 1: 启程与双岛跨越** 
  - 清晨离开 Canggu，中午前抵达 Padang Bai。
  - 午后搭乘轮渡，傍晚抵达龙目岛 Lembar。夜宿 Mataram 或 Senggigi。
- **Day 2: 穿越龙目，登陆松巴哇**
  - 上午横穿龙目岛，中午前抵达 Kayangan 港。
  - 下午滚装船登陆 Poto Tano。傍晚时分在 Kenawa 岛附近拍摄史诗级日落。夜宿西松巴哇。
- **Day 3: 西松巴哇海岸摄影日**
  - 探索 Maluk 和 Sekongkang 地区。重点拍摄热荒野里的极限海滩，寻找本地野生动物。
- **Day 4-5: 深入腹地 (公路片核心)**
  - 沿着松巴哇北部主路向东，路况逐渐变差，荒野感拉满。前往 坦博拉火山 (Mount Tambora) 山脚下或冲浪者天堂 Lakey Peak。
- **Day 6: 深度拍摄与原住民互动**
  - 在大本营及周边地区停留，进行具有深度的 Leica 人文摄影与历史记录。
- **Day 7-8: 返程阶段**
  - 开始以高机动性向西疾驰返回，可以选择不同的内陆小道。

## 💰 3. 预算规划 (Budget Allocation)

*注：行程为双人双车，印尼盾(IDR)/人民币(RMB)为参考估算，标准为中端偏紧凑（保证部分安全住宿与设备补给）*

1. **交通与轮渡 (Transport & Ferries)**:
   - 燃油: 双人双车合计约 ¥600 
   - 轮渡票 (带摩托车): Bali-Lombok 每车约 ¥80，Lombok-Sumbawa 每车约 ¥40。双车往返总计约 ¥480。
2. **住宿 (Accommodation)**:
   - 7晚住宿: 可以选择分摊双床房/标间，每晚约 ¥150-300，双人总计 = ¥1500 - ¥2000
3. **餐饮 (Food)**:
   - 日均单人 ¥100 x 8天 x 2人 = ¥1600
4. **意外与备用金 (Emergency Funds)**:
   - 2辆摩托车的轮胎快修补贴、当地带路向导费、医疗等：¥2000
5. **总预算估算**: 约 **¥6000 - ¥7000（双人总计）** (不含自有摄影器材与摩托车折旧)

## 🎒 4. 物资准备 (Equipment & Gear)

### 🏍️ 摩托与骑行装备
- **车辆**： 2 辆 **Kawasaki W175**。
- **轮胎升级**：已更换大号越野胎（前轮 3.5 英寸 / 后轮 4.0 英寸），极大提升了抓地力，足以应付松巴哇的砂石和非铺装路面。
- 半季候网眼骑行服 + 防摔护具（印尼低海拔极度炎热）
- **高品质雨衣**（热带阵雨必备，必须做到 100% 防水，保护器材）
- 工具包：便携打气筒、内胎/补胎工具组、备用火花塞、链条油。由于是复古车系，带好常用的扳手。
- 防水驮包（Drybag）与坚固的绑带（Rok Straps）。

### 📷 摄影设备组合
- **主视角与第一人称**：**Insta360 Ace Pro 2** (可挂脖以第一视角捕捉骑行反应，或挂在头盔上拍摄风景)。
- **主内容创作与Vlog**：**DJI Pocket 3** (超强手持云台相机，主要用来对着自己口播、介绍风景和记录突发事件，画质也是顶级的)。
- **航拍第三视角**：**DJI Mavic 4 Pro** (捕捉绝美壮阔的上帝视角，史诗级画面的核心)。
- **人文与静态绝杀**：**徕卡 Leica M11-P** (极致的旁轴人文拍摄体验，搭配大光圈镜头，不仅能拍摄黑白调子的路边小孩，也能拍摄高质量的夜间营地静态静帧)。
- **存储与电力**：大容量 SSD 移动硬盘，100w+ 大功率充电宝（大量耗电设备必备，部分偏远地区常停电）。

## ⛽ 5. 加油站补给与生存建议

- **油品选择**：尽量加 Pertamax (92号) 以保护发动机。
- **分布差异**：
  - Bali 和 Lombok 沿途 Pertamina (正规国营加油站) 很密集。
  - **Sumbawa 警告**：离开主城 (Sumbawa Besar, Dompu) 后，正规加油站会急剧减少。
- **"Pertamini" 玻璃瓶战术**：在松巴哇，只要油表下降到 **1/2 以下**，看到路边小卖部（Warung）门口摆着机器或玻璃瓶装的汽油（通常蓝色是普通，绿色/黄色好一点），必须立刻补满！不要抱有“下一个镇子就有正规加油站”的幻想。
- **现金**：松巴哇许多地方网络信号差，无法使用电子支付，一定要在巴厘岛或龙目岛的大城市提前取足面额为 50,000 和 100,000 的印尼盾现金。

## 📣 6. 赞助商推广与植入 (Sponsorship & Ad Placements)

> ⚠️ **核心原则**：赞助口播超过 30 秒，观众跳过率会陡峭上升。每个赞助商植入控制在 **15-25 秒**以内，用画面和字幕辅助传递信息，减少纯口播时长。

为了在旅行记录视频中自然且有效地露出赞助商，我们将赞助植入统一安排在 **Outro 段落 (15:00 前后)**——此时观众已经被内容打动，情感投入最高，广告转化效果最好。

### 1. Bad Girl Come To Bali (服装品牌) — ≤15 秒 | 视觉优先
- **品牌核心**：设计师 Dora 创立的巴厘岛辣妹T恤品牌，主打热带自由风格。
- **植入时机**：Outro 段落开头，脱骑行服换 T 恤的过程。
- **植入方式**：以**视觉展示为主**，Pocket 3 拍摄换装过程，镜头自然扫过 T 恤设计细节。口播仅一句话。
- **精简口播**：
  > "骑了一整天终于能喘口气——换上设计师 Dora 的 Bad Girl Come To Bali 透气 T 恤，整个人瞬间活过来了。来巴厘岛的话可以看看这个很酷的本地牌子。"

### 2. Ayook.id (电动摩托车品牌) — ≤20 秒 | 与环保情感线绑定
- **品牌核心**：提倡绿色出行的巴厘岛电动摩托车租赁品牌，提供两款车型。官网：[ayook.id](https://ayook.id)
- **植入时机**：Outro 结尾 VO 独白前，与"保护环境"的感悟自然衔接。
- **植入方式**：面对 Pocket 3 口播，语气融入叙事情感，不要像读广告词。
- **精简口播**：
  > "说实话，越是深入松巴哇这种原生态的地方，越能感受到保护它的重要性。这次我们骑的是汽油车，但如果在巴厘岛本地短途出行，我更推荐你试试 Ayook.id 的电动摩托——零排放，没有引擎轰鸣，在稻田和海风中你只听得到自然的声音。链接在描述栏里。"

## ⚠️ 7. 风险预案 (Risk Management)

| 风险 | 概率 | 应对措施 |
| :--- | :---: | :--- |
| **摩托车抛锚/链条断裂** | 🔴 高 | 携带备用链条扣、补胎工具 + 备用内胎；出发前记录沿途修车师傅电话（询问 Warung 老板）；`README.md` 工具包已涵盖 |
| **爆胎（碎石路面）** | 🔴 高 | 已升级越野大脚胎降低概率；携带便携打气筒 + 内胎补丁；松巴哇全程保持时速 ≤60km/h |
| **暴雨 / 山体滑坡** | 🟡 中 | 每日出发前查天气预报（离线下载 Windy App 数据）；行程预留 2 天灵活缓冲日；暴雨时就近找 Warung 避雨，不强行赶路 |
| **设备进水/损坏** | 🟡 中 | 所有设备双层防水（内层硅胶袋 + 外层 Drybag）；SD 卡分散存储（不把所有素材放一张卡）；无人机雨天严禁起飞 |
| **手机无信号** | 🟡 中 | 提前离线下载 Google Maps 松巴哇全岛地图；携带纸质地图备份；在有信号时向家人报平安 |
| **人身受伤（摔车/中暑）** | 🟠 低-中 | 全程穿戴护具（护膝、护肘、手套）；携带急救包（碘伏、纱布、止痛药、防晒霜）；记录最近医院位置：Sumbawa Besar RSUD / Dompu RSUD |
| **素材全丢** | 🔵 低 | **每日晚间**备份当日素材到 SSD 移动硬盘；关键素材（Cold Open、留存锚点、坦博拉航拍）在有 WiFi 时同步到云端 |
| **签证/通关问题** | 🔵 极低 | 三岛均在印尼境内，无需额外签证；携带护照及 KITAS/签证复印件备查 |
| **当地文化冲突** | 🔵 低 | 松巴哇以伊斯兰教为主，着装注意避免过度暴露；进清真寺周边区域需尊重当地习俗；拍摄前征得被摄者同意 |
