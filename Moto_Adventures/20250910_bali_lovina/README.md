# Canggu - Lovina

这是一个我已经拍好素材的视频项目，从巴厘岛的Canggu到Lovina去追海豚，我描述下行程经过，9月9日从canggu出发，当天到了Lovina，并找到了酒店入住，第二天早上5点出发去追海豚，那天看到了日出和许多海豚。

## 素材列表

9月9日
- 胸部视角，11:40左右，刚从家出发，骑摩托车，手机导航显示79公里，2个小时20分钟到Lovina。拍摄了Canggu出发的画面。视频有停下来佩戴雨具。
- 胸部视角，下雨了，导航显示还是1小时30分钟。我已经穿好了雨衣在骑车，镜头被雨水模糊了，马路周围是椰子树和绿色的农田。
- 胸部视角，我们停车，在便利店门口休息。
- 胸部视角，给同伴穿雨衣准备继续出发。导航显示还有1小时20分钟。
- 手持拍摄，停在便利店门口的Kawasaku摩托车，雨水滴在排气管上，冒着白烟。
- 手持拍摄，对着摩托车龙头，雨水滴在头盔和手机上，导航显示还有1小时9分钟。
- 胸部视角，下着雨，骑着摩托车走在盘山公路上上山。 海拔越来越高，越来越陡峭。
- 胸部视角，雨停了，我们已经到了山的另一面，应该是双子湖附近，准备下山了。
- 胸部视角，在罗威娜下面的破路上行驶，路面很滑， 

9月10日
- 手持拍摄，5:39，很黑，我们从酒店出发，准备骑摩托车去海边。
- 手持拍摄，5:45，到了沙滩，和老板聊天。天还没亮。
- 胸部拍摄，刚安装好相机，天刚亮，能看清海面了，船夫在准备船只。
- 胸部拍摄，我们排队坐船，船夫把船开到海中间。
- 胸部拍摄，我们上船了，船头的女孩儿在涂身体乳。
- 胸部拍摄，环绕四周，天刚破晓，我们行驶在海面上，远处有其他船只。
- 胸部拍摄，6:22，天亮了。日出了。 
- 胸部拍摄，6:42，看到了彩虹。
- 手持拍摄，很多船只靠在一起。
- 手持拍摄，船头的两个女孩儿穿着比基尼，准备下去潜水。
- 手持拍摄，6:54，看到了海豚。
- 手持拍摄，把相机放在水底拍摄，看得到船体。
- 手持拍摄，同船的大叔在潜水。
- 手持拍摄，同船的一个女孩笑脸。
- 水底视角，蓝色的海水，但是看不到海豚。
- 7:35，很多海豚，很多船只。
- 7:44，在水底拍摄到了海豚。
- 船外侧视角，很多海豚。
- 船头视角，前面是穿着比基尼的妹子，外面是海豚。
