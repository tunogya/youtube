# Gemini CLI Project Context: Tom's YouTube Scripts

This repository is a structured content management system for the YouTube channel [@tomdya](https://www.youtube.com/@tomdya), focusing on Moto Adventures, Photography, and Action Sports. It follows a strict "one folder per project" organization within theme-based directories.

## 📂 Directory Overview

- **`/Moto_Adventures/`**: Motorcycle trips, adventure routes, and moto-vlogs.
- **`/Photography/`**: Behind-the-scenes, gear reviews, and photography tutorials.
- **`/Action_Sports/`**: Skateboarding, surfing, and outdoor extreme sports.
- **`/Sponsors/`**: Information and integration scripts for brand partners (e.g., Ayook.id, Bad Girl Come To Bali).
- **`/Templates/`**: Standard templates for scripts, shot lists, and project structures.
- **`/Ideas_and_Backlog/`**: Collection of unformed inspirations and future video topics.

## 🛠️ Project Structure (The "One Folder Per Project" Rule)

Every video project resides in its own timestamped/named folder (e.g., `20260419_bali_sumbawa/`) with the following standard files:

### Key Files
- **`README.md`**: Project planning, logistics, route design, budget, and gear lists.
- **`script.md`**: The main long-form video script/narrative.
- **`short_form_scripts.md`**: Tailored scripts for TikTok, Instagram Reels, and Stories (previews, gear highlights, etc.).
- **`shot_list.md`**: Detailed list of required shots (often in Markdown or CSV).
- **`youtube_publish_kit.md`**: SEO-optimized titles, descriptions, timestamps, and hashtags for the final upload.

### Media Subdirectories
- **`/footage/`**: Raw clips and source material.
- **`/music/`**: Background music and SFX (includes `suno_ai_prompts.md` for AI-generated tracks).
- **`/thumbnails/`**: Design assets and `ai_cover_prompts.md` for cover generation.
- **`/export/`**: Final rendered versions of the video.

## 📸 Production Stack

Common gear mentioned in scripts and plans:
- **Cameras**: DJI Pocket 3 (Vlogs/POV), Insta360 Ace Pro 2 (Action/POV), DJI Mavic 4 Pro (Aerials), Leica M11-P (Human-centric/Street photography).
- **Vehicles**: Kawasaki W175 (Modified for off-road/touring).

## 💡 Usage Guidelines for Gemini

When helping with this project, please:
1.  **Maintain the "One Folder Per Project" structure**: Always group related scripts, plans, and assets within a specific project directory.
2.  **Cross-Platform Awareness**: Generate content for both long-form (YouTube) and short-form (TikTok/Reels) simultaneously.
3.  **Sponsor Integration**: Refer to the `/Sponsors/` directory for specific brand voice and key talking points when drafting scripts.
4.  **Gear-Specific Advice**: Tailor shooting suggestions based on the production stack (e.g., suggesting Leica shots for high-contrast B&W or DJI Pocket 3 for quick vlogging transitions).
5.  **Technical Specs**: Use the `youtube_publish_kit.md` format for any publishing-related assistance.
