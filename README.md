# Tom's YouTube Scripts

这里是 Tom 的 YouTube 视频拍摄和内容创作的脚本库。

▶️ **YouTube 频道**: [@tomdya](https://www.youtube.com/@tomdya)

## 频道定位
本频道主要聚焦于：
- 🏍️ **摩托车探险** (Moto Adventures)
- 📷 **摄影与记录** (Photography & Filmmaking)
- 🛹 **滑板等极限运动** (Action Sports)

这个仓库用于管理和存放视频制作过程中的所有文案、脚本、大纲以及灵感库。

## 目录结构
为保持项目结构的清晰，本仓库采用**主题系列**作为一层目录分类。每个具体的视频项目将在对应的系列目录下建立独立的文件夹进行管理（“一案一档”原则）。

- `/Moto_Adventures/` —— 摩托车骑行、探险路线、摩旅vlog
- `/Photography/` —— 摄影作品幕后、器材评测、拍摄技巧
- `/Action_Sports/` —— 滑板、冲浪等户外极限运动系列
- `/Templates/` —— 通用的分镜表模板、脚本骨架、视频大纲模板
- `/Ideas_and_Backlog/` —— 未成型的灵感碎片收集箱、未来的备选视频选题

### 📝 工作流示例：如何筹备一个新视频？

假设你要拍一期在巴厘岛骑自行或摩托车环岛的视频，步骤如下：

1. 进入对应的分类目录 `/Moto_Adventures`。
2. 创建一个新的视频项目文件夹，建议命名附带日期或标号，例如：`202604_bali_coast_ride/`。
3. 在这个文件夹内创建你的视频脚本（如 `script.md`）和拍摄清单（如 `shot_list.csv`）。
4. 所有的参考图片、地图截图也都归档在这个文件夹内。
